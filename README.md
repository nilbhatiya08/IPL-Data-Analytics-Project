# 🏏 IPL Dataset Analysis using Pandas

## 📌 Project Overview

This project contains a comprehensive analysis of the Indian Premier League (IPL) dataset using Python and Pandas.

The notebook covers multiple levels of data analysis tasks ranging from beginner to advanced, helping learners strengthen their Pandas skills through real-world cricket data.

---

## 📂 Dataset

Dataset Used: `ipl_dataset.csv`

The dataset contains information such as:

- Season
- City
- Teams
- Toss Winner
- Match Winner
- Venue
- Player of the Match
- Win Margin
- Toss Decision
- D/L Method

- Python

- Jupyter Notebook
